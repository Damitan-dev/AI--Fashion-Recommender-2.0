# AI Fashion Recommender V2

An upgraded AI-powered fashion recommendation system.
Built with Python, TensorFlow, and Streamlit.

## What is Improved Over V1
- More accurate recommendations using EfficientNetB0
- Fuzzy product search — handles typos and partial names
- Clean interface showing product images and categories
- Separate sections for similar and complementary items

## Goal
Customer types a product name → product image appears
→ similar items shown with categories
→ complementary items shown with categories

## Build Plan
- **Month 1:** Data cleaning, preparation and
  fuzzy product search engine
- **Month 2:** Accurate visual similarity engine and
  complementary recommendation engine
  using EfficientNetB0 and cosine similarity
- **Month 3:** Clean Streamlit interface with product
  images and categorised results,
  fine tuning and deployment

## Tech Stack
- Python, TensorFlow, Scikit-learn
- EfficientNetB0, Cosine Similarity, RapidFuzz
- Streamlit, Pandas, Pillow
